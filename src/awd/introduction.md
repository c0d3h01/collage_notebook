# Introduction to AWD

Welcome to the **Advanced Web Design (AWD)** practical reference. This collection compiles **45 practical exercises** designed to guide you through the fundamental and advanced aspects of web development.

The material is structured to provide hands-on experience with:

* [cite_start]**HTML5 Structure:** Creating semantic web pages using lists, tables, nested structures, hyperlinks, and comprehensive forms[cite: 1, 6, 9, 25, 26].
* [cite_start]**CSS3 Styling:** Mastering visual design through selectors, combinators, layouts, positioning (z-index), and responsive techniques[cite: 27, 37, 42].
* [cite_start]**JavaScript Interactivity:** Implementing client-side logic for form validation, mathematical calculations (Fibonacci, Factorial, Armstrong numbers), array manipulation, and DOM interaction[cite: 48, 52, 60, 70].

Each practical includes the complete source code and a preview of the expected output, serving as a quick reference for students and developers alike.
